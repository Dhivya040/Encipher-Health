Common Image Formats

JPEG
Used mainly for photographs and realistic images. It offers good compression but may lose some quality
PNG
Supports transparency and uses lossless compression, making it ideal for web graphics and logos
SVG
Stands for Scalable Vector Graphics. It is based on XML and remains clear at any size, perfect for icons and illustrations
